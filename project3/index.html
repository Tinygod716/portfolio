Utopia Planet

In this project, I will use html, css and javascript to create a virtual universe that shows the infinite possibility of earth in the distant future. Inspired by a planet simulation game, this project aims to create an interesting earth simulation to predict possible human action towers on those planets..

Role Web Designer
Timeline 4 WEEKS
Tools Html / Css / Javascript

FIG 01A. Stacked Concept Cards

"Utopia or Anti-Utopia?"

Each planet is imagined as a speculative world shaped by a single core idea. Their differences are not only visual, but also environmental and social. By exploring them one by one, viewers can reflect on how technology, emotion, and systems of control might reshape Earth in the future.
